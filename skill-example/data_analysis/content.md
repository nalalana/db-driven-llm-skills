# 数据分析技能

## 核心能力

1. **数据清洗**
   - 识别和处理缺失值
   - 异常值检测和处理
   - 重复数据清理

2. **统计分析**
   - 描述性统计
   - 相关性分析
   - 假设检验

3. **数据可视化**
   - 创建图表、直方图、散点图、热力图等
   - 数据趋势分析

4. **报告生成**
   - 撰写清晰的数据分析报告
   - 包含关键发现和建议

## 工作流程

1. 理解数据需求和分析目标
2. 加载和检查数据质量
3. 执行数据清洗和预处理
4. 进行统计分析
5. 创建可视化图表
6. 总结关键发现并生成报告

## API 调用说明

### 获取数据统计

调用 `get_data_statistics` API 获取数据统计信息：

- **参数**：
  - `data_id` (必需): 数据标识符
  - `format` (可选): 返回格式，默认 json

- **返回**：
  - 统计摘要
  - 关键指标
  - 数据质量报告

## 常用工具

- pandas: 数据处理
- numpy: 数值计算
- matplotlib/seaborn: 数据可视化
- scipy: 统计分析

## 最佳实践

- 始终检查数据质量
- 使用适当的统计方法
- 创建清晰、有意义的可视化
- 用通俗易懂的语言解释技术发现
