# 代码审查技能

## 审查维度

1. **代码质量**
   - 可读性和可维护性
   - 代码风格和规范
   - 命名约定
   - 代码结构组织

2. **性能优化**
   - 算法复杂度分析
   - 资源使用效率
   - 缓存策略
   - 并发处理

3. **安全性**
   - SQL注入防护
   - XSS攻击防护
   - 认证和授权
   - 敏感数据处理

4. **最佳实践**
   - 设计模式应用
   - 错误处理
   - 测试覆盖率
   - 文档完整性

## 审查流程

1. 理解代码功能和上下文
2. 检查代码结构和组织
3. 识别潜在问题和改进点
4. 提供具体的改进建议
5. 评估整体代码质量

## 输出格式

- 问题分类（严重/中等/轻微）
- 具体位置和代码片段
- 问题描述和影响
- 改进建议和示例代码
